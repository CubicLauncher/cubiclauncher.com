# Cómo Contribuir

¡Nos alegra que quieras ayudar a mejorar CubicLauncher! Somos un proyecto impulsado por la comunidad y cualquier ayuda es bienvenida.

## ¿Cómo puedes ayudar?

### 1. Reportar errores
Si encuentras un bug o un comportamiento inesperado:
- Abre un **Issue** en nuestro [repositorio de GitHub](https://github.com/CubicLauncher/CubicLauncher/issues).
- Describe los pasos para reproducir el error.
- Adjunta los logs de la [consola](#consola) si es posible.

### 2. Sugerir funciones
¿Tienes una idea genial? 
- Abre una discusión o un issue etiquetado como `enhancement`.
- Explica por qué la función sería útil para el resto de usuarios.

### 3. Desarrollo de código
Si quieres meter mano al código:
1. Haz un fork del repositorio.
2. Crea una rama para tu mejora (`git checkout -b feature/nueva-funcion`).
3. Realiza tus cambios y asegúrate de que el proyecto compile (`./gradlew build`).
4. Abre un **Pull Request** explicando qué has cambiado.

### 4. Traducciones
Queremos que CubicLauncher llegue a todo el mundo. Si hablas varios idiomas:
- Ayúdanos a traducir los archivos en la carpeta `src/main/resources/lang/`.

### 5. Documentación
¡Incluso ayudando con esta wiki estás contribuyendo! Puedes editar estas páginas directamente en GitHub.

## Requisitos técnicos

Para desarrollar en CubicLauncher necesitarás:
- **Java 21 JDK** o superior.
- **IntelliJ IDEA** (recomendado) o cualquier IDE compatible con Gradle.
- Conocimientos básicos de JavaFX para la interfaz.
