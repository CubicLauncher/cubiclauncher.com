# Consola de CubicLauncher

La **Consola** es una herramienta esencial para el monitoreo del juego y la depuración de errores. Proporciona una vista en tiempo real de los registros (logs) tanto del launcher como de Minecraft.

## ¿Cómo abrir la consola?

Puedes habilitar la consola desde la configuración general activando la opción **"Consola de errores"**. Una vez activa, se abrirá una ventana adicional al iniciar el juego.

## Características de la Consola

- **Registro en tiempo real**: Visualiza la salida de Minecraft mientras se ejecuta.
- **Detección de errores**: Resalta automáticamente advertencias y errores críticos para facilitar su identificación.
- **Exportación de Logs**: Permite copiar o guardar el registro para compartirlo con el equipo de soporte.
- **Comandos**: En futuras versiones, permitirá la interacción directa con la instancia del juego.

## Depuración de errores comunes

Si el juego se cierra inesperadamente (Crash), la consola mostrará la causa raíz. Los errores más comunes suelen ser:

1.  **OutOfMemoryError**: Java se ha quedado sin memoria asignada. Prueba aumentar la RAM en la [Configuración](#configuracion).
2.  **Incompatible JRE**: Estás intentando ejecutar una versión de Minecraft con una versión de Java incorrecta.
3.  **Missing Libraries**: Archivos corruptos o descargas incompletas.
